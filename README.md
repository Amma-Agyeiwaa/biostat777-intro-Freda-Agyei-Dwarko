## Project Overview
This repository hosts a personal Quarto website created for **Biostat 777 ‚Äì Introduction to Data Science in Public Health**.  
The website includes:

1. **Home Page** ‚Äì A brief introduction and photo  
2. **About Page** ‚Äì Biography, education, and professional experience  
3. **Example Analysis Page** ‚Äì A reproducible data analysis using the **Gapminder** dataset to explore the relationship between life expectancy and GDP per capita.

The project demonstrates the use of:
- Quarto for reproducible scientific publishing  
- R programming and the *tidyverse* for data wrangling and visualization  
- Git + GitHub + GitHub Pages for version control and web deployment  

---

## Technical Details
- **Built with:** [Quarto](https://quarto.org) v 1.5 +  
- **Language:** R  
- **Key Packages:** `tidyverse`, `gapminder`, `janitor`, `forcats`, `scales`  
- **Output Directory:** `docs/` (for GitHub Pages)  
- **Theme:** *flatly* (Bootstrap)  
- **Deployment:** GitHub Pages  

To rebuild locally:
```bash
quarto render
```
